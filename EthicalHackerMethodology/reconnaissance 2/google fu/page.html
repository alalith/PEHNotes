Google Search Operators

Example: tesla

Query 1: site:tesla.com (searchs only tesla.com domain)

Query 2: site:tesla.com -www (takes out main subdomain)

Query 3: site:tesla.com -www -ir (takes out main and ir subdomain subdomain)

Query 4: site:tesla.com filetype:pdf (finds all pdfs in tesla.com domain)

Popular filetypes to search: pdf, xlsx, csv
