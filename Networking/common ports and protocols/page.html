TCP:
  FTP (21) (File Transfer Protocol) - transfer files between servers
  SSH (22) (Secure SHell) - Encrypted access to server
  Telnet (23) - Unencryptted access to server
  SMTP (25) - Email
  DNS (53) (Domain Name System) - resolves ip of domain names
  HTTP (80) / HTTPS (443) (HyperText Transfer Protocol) - Web traffic
  POP3 (110) - Email
  SMB (139 + 445) - Originally 139, later version use 445. File shares. Also called Samba
  IMAP (143) - Email

UDP:
  DNS (53)
  DHCP (67, 68) - Assigns device a random private IP address for a leased time. Opposite is static
  TFTP (69) (Trivial File Transfer Protocol) - FTP over UDP
  SNMP (161) (Simple Network Management Protocol)
